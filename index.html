Urology Consent Help & Guide

1. Out-of-the-Box Fallback System

If you don't have page1.jpg or page2.jpg uploaded to your server/repository, the app will automatically detect it and draw a clean, professional, high-resolution document outline dynamically in the PDF. This allows you to print immediately or test the app anywhere.

2. Uploading Custom Images

To use the actual official Saveetha Consent Forms (English or Tamil versions), simply save your forms as JPGs and name them exactly:

  • English version: page1.jpg and page2.jpg
  • Tamil version: page1_tamil.jpg and page2_tamil.jpg

Upload these files to the same directory on GitHub or your web server where this index.html is hosted.

3. Calibration & Scanning offsets

Since scan dimensions vary (e.g. from camera captures or scanners), use the Calibration tab. Simply slide the markers to position where the text sits. The values are percentages of the image dimensions to work on both high-resolution scans and light phone snapshots. Stamped signatures will follow these boundaries perfectly.

4. Data Security (No-Cloud/HIPAA Compliant)

This application is 100% client-side. The PDF is generated entirely in your phone's memory. No patient names, PIDs, medical details, or signatures are sent over the network or saved to any cloud database. When you close or refresh the tab, all inputs are wiped instantly.
